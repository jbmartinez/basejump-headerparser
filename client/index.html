API Basejump: Request Header Parser Microservice

What does this do?

  • Send a request and it will respond with the IP address, preferred language and operating system for the browser.

Example usage

Requests

https://name.herokuapp.com/api/whoami

Output

{
  "ipaddress":"10.0.2.2",
  "language":"en-US",
  "software":"Windows NT 10.0; WOW64"
}
